운영 메모

기사 추가 news-index.json에 새 항목을 추가하고, 필요 시 articles/news 폴더에 개별 본문 JSON을 생성합니다.
이미지 연결 assets/images/news에 업로드한 이미지를 image 경로에 연결하면 목록과 상세에 자동 반영됩니다.
상세 출력 article 페이지는 id 값을 읽어 개별 JSON 본문과 legacy 기사 데이터를 순차적으로 확인해 자동 출력합니다.
