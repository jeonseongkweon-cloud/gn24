GN24 NEWSROOM

기사 데이터를 추가하면
목록과 상세가 자동으로 연결됩니다

GN24는 data/news.json에 기사 데이터를 누적하는 방식으로 운영됩니다. 이미지 업로드와 기사 데이터 추가만으로 뉴스룸 목록과 상세 페이지가 자동 구성되도록 설계했습니다.

운영 방식
이미지 업로드 → news.json 추가 → 자동 목록/상세 출력
중점 분야
생활 안전 · 청소년 교육 · AI·혁신기술 · 무도·스포츠
핵심 장점
기사마다 HTML 새로 만들지 않고 데이터만 누적
홈으로
전체 기사 0건

운영 메모

기사 추가 news.json에 새 객체 1개를 추가하면 됩니다.
이미지 연결 assets/images/news 폴더에 업로드한 파일 경로를 image에 넣습니다.
상세 출력 article/index.html이 id 값을 읽어 자동으로 기사 본문을 출력합니다.
